# MOMNT

> Scan. Upload. Relive the moment.

MOMNT is a QR-based real-time event photo sharing platform that allows attendees to instantly upload photos to a shared event album — without downloading an app or creating an account.

Built to make event memory collection effortless, interactive, and modern.

---

## ✨ Features

### 📸 Instant Photo Upload
- Scan QR code
- Open upload page
- Upload photos instantly

No app. No login. No friction.

---

### ⚡ Real-Time Event Experience
- Photos can appear live during events
- Fast and simple interaction
- Built for high participation

---

### 📱 Mobile-First Design

- Modern startup-style interface

## 🧠 The Idea

Event photos are usually:
- Scattered across devices
- Shared inconsistently
- Lost after the event

Existing solutions often require:
- App downloads
- Logins
- Extra effort

MOMNT solves this by creating a frictionless experience:

```text
Scan → Upload → Done
```
